## 항상 지키는 원칙
- 구현 안 한 기능 **발표 금지**, 근거 없는 수치 금지.
- **판정·계산은 결정론적 코드**, AI는 이해·대화·값 추출.
- 발표 시간의 **50%+ 를 데모+킥**에. 시간 부족 = 단계 생략❌ → 기능 범위 축소⭕.
- **LLM 키 = 앱 런타임 직접 입력**(⚙️·⋯ → 키 모달), 키 없으면 폴백. 코드/저장소에 키 금지.
- 모바일 반응형 필수. **반응형 ≠ 앱**(데스크톱 크롬 걷어내기).
