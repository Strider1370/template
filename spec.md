# spec.md — 스펙 문서

> **Stage 03 산출물. 확정된 내용만.** (후보·탈락안·리서치 원문·발표 스크립트는 넣지 않는다.)

> 아래 16개 헤딩은 Gate(`npm run gate:spec`)가 문자열로 검사하므로 **헤딩을 바꾸지 마라.**
> 각 헤딩 밑의 HTML 주석은 작성 안내다. 채울 때 주석은 지우고 실제 내용만 남겨라.
> ⚠️ 개인정보: 실제 개인정보를 데모 입력값으로 쓰지 마라. **가상 페르소나 / 익명 fixture**만 사용한다.
