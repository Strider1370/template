# 해커톤 진행 체크리스트 (일반 프로세스 — 인쇄용 1장)

> 주제·코드·특정 인프라 무관. "이렇게 일한다" 방법만. 현장에서 보고 LLM에 지시·직접 작성용.
> 결과물은 당일 시제 받고 새로 제작. 신규 레포 + 점진 커밋(과정 보이게) + 쓴 템플릿/데이터/라이브러리 출처 제출.

## 항상 지키는 원칙 (cross-cutting)
- 구현 안 한 기능 **발표 금지**(정직성). 근거 없는 수치 만들지 말기.
- **판정·계산은 결정론적 코드**, AI는 이해·대화·값 추출만.
- 발표 시간의 **50%+ 를 데모+킥(차별점)** 에.
- 시간 부족 = 단계 생략 ❌ → **기능 범위 축소** ⭕ (데이터 말고 화면 수를 줄여라).
- 데이터는 **모집단 전수, 표시는 일부**.
- 모바일 반응형 필수. **반응형 ≠ 앱** → 모바일에선 데스크톱 크롬(GNB·큰 히어로) 걷어내고 핵심 입력을 첫 화면에.
- **LLM 키 = 런타임 직접 입력**(설정/더보기 ⚙️·⋯ 아이콘 → 키 모달, 메모리만). 키 없으면 폴백 동작.

## 13단계
- **00 접수** — 주제·제약·발표시간·구현시간 확인. 가벼운 방향 발산(확정 X).
- **01 리서치(병렬)** — 해외사례 ≥3·데이터 가용성·"킥(차별점)" 후보. 출처 연결. (탐색 폭 = 품질)
- **02 인사이트 선택 ★승인** — 후보 ≥5 발산 → **레드팀(깐깐한 심사위원으로 자기공격: 흔한앱 아닌가/AI 왜 필요/데모로 보이나/기억될 한문장 있나)** → 1개 확정. concept = 한문장피치·킥·Wow·마지막문장.
- **03 스펙** — 데모 시나리오·Wow Moment·**화면 입출력(필드/결과) 구체 명시**. "4시간 현실성"으로 범위 컷. (음성 쓸지 여기서 결정)
- **04 계획** — 작업분해·파일소유권·폴백. 재사용 자산 매핑(같은 것 새로 만들지 말기).
- **05 빌드** — 핵심경로부터·fixture로. **작으면(파일<3·단일페이지) 단독이 더 빠름**(과병렬 금지).
- **06 통합** — 핵심경로 끝까지 연결 + manifest(기능 상태: implemented/mocked/fallback/dropped). 통합할 게 없으면 05에 흡수.
- **07 데모검증** — **사람이 눈으로 작동 확인**(가장 빠름) + **단계별 스크린샷 백업**(라이브 실패 대비). 영상 X.
- **08 대본** — 답먼저→문제→인사이트→데모→클로징. 미구현 발표 금지. Q&A = **심사위원 공격질문 10개 + 답**.
- **09 발표생성** — 1슬라이드 1메시지, 제목 = 결론형 주장. **화면 카피는 사람이**(AI초안 → 마음에 안 드는 것만 덮어쓰기). 톤: 금지어(혁신적·게임체인저·seamless…)·**첫 문장은 사람이**·발표장 어투.
- **10 발표검증** — 자동 overflow 체크 + **사람 눈으로 이상한 슬라이드만 지목** → 수정.
- **11 리허설 ★승인** — 낭독 시간 측정(시간 내), 라이브↔스크린샷 전환 점검, **최종 승인**.
- **12 패키지** — 배포 URL + (선택)APK + 출처·라이선스 제출물.

## 기술 빠른참조 (일반 방법)
- **배포(https)**: 서버에 앱 상주(`next start` 등) + nginx 리버스프록시 + certbot(Let's Encrypt). `/api/*` 있으면 정적 export 불가.
- **APK**: Capacitor 래퍼, `server.url = 배포 https URL`(cleartext 불필요). 빌드는 **서버/빌드머신에서**(JDK21+Android SDK, gradlew). 폰 설치 `adb install -r`, 시연 **scrcpy USB 미러링**(무선 페어링은 폴백). 삼성폰은 **삼성 USB 드라이버** 필요. WebView 마이크는 `RECORD_AUDIO`+`MODIFY_AUDIO_SETTINGS`+권한 override.
- **음성(쓸 때)**: `gpt-realtime-2`(reasoning low) · WebRTC · **strict 함수호출로 폼 채움** · `semantic_vad`(eagerness low, 어르신) · 값 read-back 확인 · **자격 판정은 코드**.
